Сервис проверки

Сервис позволяет автоматически проверять корректность предоставленных сведений
Для работы сервиса достаточно загрузить файл с предоставленными данными
Возможные варианты ответа сервиса:

  • Зелёный свет - проблем не выявлено
  • Жёлтый свет - рекомендуется подтверждение со стороны оператора
  • Красный свет - выявлены серьёзные расхождения. Не рекомендуется одобрение

Сервис позволяет сформировать отчет либо выгрузить данные в формате xlsx
